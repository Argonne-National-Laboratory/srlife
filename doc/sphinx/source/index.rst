srlife: fast estimation of thermal receiver structural life
===========================================================

srlife is a python package aimed at assessing the expected life of high
temperature metallic solar receivers and other components.  The 
:any:`overview` section describes the package objectives, organizations, 
and assumptions, the :any:`installation` section provides installation
instructions, and :any:`tutorial` provides a tutorial on how to run
the package for a sample analysis.
